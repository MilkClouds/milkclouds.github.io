생략


## 설명

`left, right`를 적당히 설정하여 이분 탐색을 한다.
`mid` 에서의 값이 정답보다 큰 지 작은지 확인을 할 때는, 팩토리얼 0의 개수를 셀 때 인수 2의 개수는 충분하지만 5의 개수에 따라 0의 개수가 바뀌는 성질을 이용했다.
즉 `f(n)`은 `n!`에서 5가 몇번 곱해져 있는지 셈으로써 0의 개수를 셌다.


### 소스  

```python
n=int(input())
l,r=0,10**9

def f(n):
	r=0
	while(n):
		r+=n//5
		n//=5
	return r

while(l+1<r):
	m=(l+r)//2
	if f(m)>=n: r=m
	else: l=m

print(r if f(r)==n else -1)
```
