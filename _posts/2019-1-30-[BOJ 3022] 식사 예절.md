## 문제
[https://www.acmicpc.net/problem/3022](https://www.acmicpc.net/problem/3022)

## 사용 알고리즘
해쉬 맵

## 시간 복잡도
`O(N)`


## 설명
해쉬 맵에 [] 연산자가 있는게 참 편하다고 생각한다.


### 소스  


```c++
#include <bits/stdc++.h>
using namespace std;

int N,cnt;
string st;
unordered_map<string,int> mm;
int main(){
	cin>>N;
	for(int i=0;i<N;i++){
		cin>>st;
		if(i-mm[st]<mm[st])cnt++;
		mm[st]++;
	}
	cout<<cnt;
}
```
