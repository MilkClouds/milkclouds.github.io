## 문제
[https://www.acmicpc.net/problem/11375](https://www.acmicpc.net/problem/11375)

## 사용 알고리즘
네트워크 플로우  
이분 매칭

## 시간 복잡도
`O(VE)`


## 설명
전형적인 이분 매칭 문제입니다.


### 소스  

```python
import sys
input=sys.stdin.readline
N,M=map(int,input().split())
flow=['']
for i in range(N):
    flow.append(list(map(int,input().split()))[1:])

rev=[0]*1005
def dfs(n):
    if visited[n]: return 0
    visited[n]=1
    for i in flow[n]:
        if not rev[i] or dfs(rev[i]):
            rev[i]=n
            return 1

ret=0
for i in range(N):
    visited=[0]*1005
    if dfs(1+i):ret+=1
print(ret)
```
